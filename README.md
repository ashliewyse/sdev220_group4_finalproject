# SDEV 220: Group 4 Microwave Simulation

## Project Overview
This project is a Command Line Interface (CLI) simulation of a microwave oven. It demonstrates object-oriented programming (OOP) principles by separating the logic into dedicated classes for the Microwave, Timer, and User Interface.

## Module 3 Requirements Met
* **Classes & Objects:** Utilized for core microwave functionality and timer management.
* **Dictionaries:** Implemented to store and recall cooking presets (e.g., Popcorn, Beverage).
* **Real-Time Timer:** A functional countdown timer that supports pausing and stopping.
* **Completion Notification:** A simulated "Done" signal (Visual "Ding") triggered upon timer completion.
