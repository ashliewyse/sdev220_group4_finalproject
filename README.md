# SDEV220 Group 4 – Microwave Simulation (CLI Draft)

This repository contains the initial Command-Line Interface (CLI) draft for the Group 4 final project.

## Features Implemented
- Menu-based CLI
- Power on/off handling
- Cook time input validation
- Start, pause, resume, stop logic
- Basic state validation messaging
- Status display

## How to Run

Clone the repository and run:

```bash
python microwave_cli.py
